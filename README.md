## How to Install
### Windows / MacOS
```
git clone https://github.com/OkayJerry/Time-DomainAnalysis.git
cd Time-DomainAnalysis
pip install -r requirements.txt
```

### Linux (Debian Bookworm)
```
git clone https://github.com/OkayJerry/Time-DomainAnalysis.git
sudo apt install python3-pyqt6
sudo apt install python3-pyqtgraph
```
